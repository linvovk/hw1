## Описание проекта

**Тема проекта:** Современные вызовы в финтех-менеджменте. Особенности финтех-проектов vs классических IT-проектов.

**Цель проекта:** На основе анализа ключевых особенностей и вызовов финтех-проектов разработать сравнительную модель и реализовать прототип.


## Структура репозитория:

### Часть 1. Аналитическо-сравнительная (теоретическая)
В папке `/analysis` представлен сравнительный анализ финтех-проектов и классических IT-проектов по ключевым критериям.

### Часть 2. Программная реализация (прототип)
В папке `/src` находится Python-скрипт модуля предварительной проверки транзакций (fraud detection). Скрипт демонстрирует, как регуляторные требования встраиваются в код.

### Часть 3. Проектное управление
В папке `/docs` описана структура  финтех-проекта: стейкхолдеры, требования, риски и их митигация.

### Пояснительная записка
В файле Note.md содержатся основные выводы по всем трем частям проекта.
